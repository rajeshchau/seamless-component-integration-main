

# Landing Page Website Plan

## Overview
Build a modern, single-page landing website by integrating all four provided components into one cohesive scrollable page.

## Page Structure (Top to Bottom)

### 1. Navigation Bar
- **Kinetic Navigation** component at the top with animated full-screen menu overlay
- GSAP-powered animations for menu open/close transitions
- Hover effects with abstract background shapes
- Menu links: About Us, Our Work, Services, Blog, Contact Us

### 2. Hero Banner
- Full-screen hero section with a stunning background image (replaced with working Unsplash space/cosmos imagery)
- Responsive top navigation bar with mobile hamburger menu
- Animated badge announcement, large headline, and description text
- Two CTA buttons: "Book Your Journey" and "Watch Launch"
- Partner logos section at the bottom of the hero

### 3. Interactive Links Section
- Hover-animated link list showcasing key pages (Services, Team, Projects, Careers, Playground)
- Mouse-tracking image preview that follows cursor on hover
- Smooth Framer Motion animations

### 4. Testimonials Section
- Auto-scrolling testimonial cards in three columns
- User photos, names, roles, and quotes
- Dark mode toggle included for the whole site

## Design & Styling
- Dark, modern aesthetic with space/cosmos theme from the hero
- Smooth animations throughout (GSAP + Framer Motion)
- Fully responsive across desktop, tablet, and mobile
- Custom CSS variables for consistent theming
- Google Fonts: Inter (sans-serif) and Instrument Serif

## Dependencies to Install
- `gsap` — for kinetic navigation animations
- `motion` — for interactive hover links and testimonials
